🚀 1. 프로젝트 한 줄 소개

“기록”을 넘어 “해석”까지 제공하는 데이터 기반 개인 금융 인텔리전스 서비스

CSV/Excel 지출 데이터를 업로드하면
✔ 자동 전처리
✔ 카테고리 분석
✔ 소비 패턴 시각화
✔ GPT 기반 AI 인사이트 생성
✔ 맞춤형 예산 추천

까지 자동으로 수행하는 Streamlit 기반 웹 대시보드입니다.

🎯 2. 문제 정의 (Problem Statement)

기존 가계부 서비스는 다음 한계를 가집니다:

수기 입력 중심 → 사용자 피로도 증가

단순 기록 제공 → 해석 부재

개인화 부족 → 행동 변화 유도 어려움

본 프로젝트는 다음 질문에서 출발했습니다:

“사용자의 소비 데이터를 자동 해석하고, 실제 행동 변화를 유도할 수 없을까?”

🧠 3. 해결 전략 (Solution Approach)
1️⃣ 데이터 파이프라인 설계
사용자 업로드
      ↓
데이터 클리닝 (Pandas)
      ↓
카테고리 분석
      ↓
통계 계산 (비율/평균/편차)
      ↓
GPT 프롬프트 엔지니어링
      ↓
인사이트 및 예산 가이드 생성
🏗️ 4. 아키텍처 설계
UI Layer (Streamlit)
│
├── Sidebar (입력/설정)
├── KPI Metrics
└── Plotly Charts
│
Service Layer
├── Data Loader
├── Expense Analyzer
├── AI Insights (GPT)
└── Report Generator
│
Model Layer
└── Expense Data Object

✔ UI / 분석 로직 / AI 계층 분리
✔ 유지보수 및 확장성을 고려한 구조 설계
✔ 서비스 단위 모듈화

🛠️ 5. 기술 스택
영역	기술

구조 설계	Layered Architecture
